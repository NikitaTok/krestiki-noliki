# Крестики-нолики на Python в консоли

Классическая консольная игра "Крестики-нолики" для двух игроков с интуитивным управлением и проверкой победных комбинаций.

### Игровой процесс
- **Два игрока** (X и O)
- **Пошаговая система** с чередованием ходов
- **Валидация ходов** (проверка занятости клеток)
- **Определение победителя** по всем правилам
- **Ничья** при заполнении всех клеток

### Интерфейс
- **Чистая консоль** без лишних зависимостей
- **Простая навигация** по координатам
- **Цветные сообщения** (опционально)
- **Отображение игрового поля** после каждого хода

### Функции
- **Проверка победы** по строкам, столбцам и диагоналям
- **Подсчет ходов** для определения ничьи
- **Защита от некорректного ввода**
- **Возможность переиграть** без перезапуска

### Требования
- Python 3.8 или выше
- Терминал/командная строка

### Установка и запуск

# Клонируйте репозиторий
git clone https://github.com/ВАШ_ЛОГИН/krestiki-noliki.git
cd krestiki-noliki

# Запустите игру
python krestiki-noliki.py
